http протокол прикладного уровня. клиент серверный протокол - набор правил передачи информации. Hyper text
pransfer protocol. 
C помощью протокола мы регламинтируем как наша информация будет передаваться
в сети.

Http  состоит:
- из основной части ( то что мы хотим передать) 
- из хедера - заголовка - то как основная часть должна передаваться 

http request 
1) версия протокола
2) указывается метод
3) версия протокола
4) хост машина (сервер)
5) хедер

http respons
1) версия протокола
2) статус код)
3) дата отправки ответа
4) инфа о сервере 
5) хедер 

Отличия http и https 
https - зашифрованный протокол , инфа попадает на сервер в зашифрованном виде
ssl  сертификат, подтверждающий секьюрность
